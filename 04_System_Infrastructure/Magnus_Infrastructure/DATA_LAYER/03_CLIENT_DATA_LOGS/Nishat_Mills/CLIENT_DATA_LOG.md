## Engagement Summary

Nishat Mills was the first Magnus dossier. It established the baseline methodology and generated the first 7 Playbook lessons (L1-L7). As a PSX-listed company, data availability was significantly better than private companies.

## Key Lessons Generated (Now in Playbook)

| Lesson | What Happened | Impact |
|--------|-------------|--------|
| L1 | NDI was just a number — readers couldn't see the work | Added visible Claim Ledger |
| L2 | MSRI was a "magic number" — no transparency | Added full MSRI Appendix |
| L3 | One unsourced claim killed trust | Added trace chains for all claims |
| L4 | Predictions were informal | Created formal Prediction Registry |
| L5 | "What Would Change My Mind" built more trust than certainty | Made it standard |
| L6 | Quarterly margins used different definitions than annual | Added explicit definition notes |
| L7 | Other income needed decomposition | Flagged for V2 improvement |
