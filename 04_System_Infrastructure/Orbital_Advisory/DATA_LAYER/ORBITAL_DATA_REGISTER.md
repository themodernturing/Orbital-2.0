# ORBITAL MASTER DATA REGISTER

> Every number in every report must trace back to this register.
> If a data point is not listed here, it cannot be used with "VERIFIED" confidence.

**Last Updated:** 2026-03-01
**Maintained By:** Orbital Intelligence System

---

## How to Use This Register

1. Before using any number in a report, check if it exists here
2. If it does → use the confidence level assigned
3. If it doesn't → add it here first, validate it, then use it
4. Every row must have a source and at least one cross-validation reference

---

## Confidence Definitions

| Tag | Meaning | Minimum Requirement |
|-----|---------|-------------------|
| **VERIFIED** | Confirmed by 2+ independent sources | Official + international cross-ref |
| **ESTIMATED** | Single credible source, no cross-check | One official source only |
| **PROXY-BASED** | Inferred from related indicator | Labelled as indirect measure |
| **HYPOTHESIS** | Logical inference, no hard data | Requires internal data to confirm |
